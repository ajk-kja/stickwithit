How it shows up in rehearsal

A random chop loads the moment the page opens, so rehearsal brains get an instant prompt. Tap the Drums button to spin a new idea, press Pattern to swap RLK orchestrations, and use the tempo plus/minus and note-count buttons to stretch or shrink the phrase without diving into menus. Now you don't have to throw change on the floor to come up with a cool linear drum fill.

What carried over from Pattern Generator

  • Thumb-first writing: The four-lane matrix stays intact so you can sketch licks with your phone hand while the other keeps time.
  • Instant auditioning: Every toggle retriggers the loop with no modal dialogs or save prompts.

What changed for the web

The Android experience assumed a single user. The web version leans into collaboration and quick sharing, so every tweak is visible and sendable from a single tab.

  • Shareable URLs: Each configuration serializes into the link, meaning a 16-step chop can be dropped into Discord with zero uploads.
  • Mobile layout: Buttons are spaced for thumbs so you can run the grid mid-commute without a stylus or the butt of a stick.
  • Tempo & note controls: Dedicated buttons adjust BPM and the number of notes so you can stretch or trim an idea before sending it to the band.
