The Gigstrella: Locking the Edges of Every Song

Published February 24, 2026 · Practice log

“Setlist Sprint” was the working title back when this was just a hackathon board. These days the app lives on shows because it does one thing well: it helps drummers nail the moments the crowd actually remembers. The new name, The Gigstrella, leans into that mission—stick the intro, stick the outro, and stay calm in between.

Using AB/CD loops like a safety net

The workflow is simple: drag the latest board mix into the browser, drop marker A where the count-in lands, B at the top of verse one, C at the start of the outro vamp, and D where the band buttons the song. With AB active, the app keeps spitting the intro back until the pickup feels automatic. Flip to CD and it loops the ending tag so players can shed stickings over a pad without touching a DAW.

Show-night triage

On gig night the same project becomes a quiet stage manager. If the singer cuts the third tune, the drummer (or MD) types the new order and The Gigstrella immediately recalculates runtime based on each song’s BPM and note count. No spreadsheets, no head math—just a quick glance before calling the next number.

Rig Notes

  • AB/CD loop states, markers, and BPM settings live in IndexedDB so I can rehearse offline on an iPad.
  • Manual + auto BPM detection share the same UI; detected tempos land first, but I can type in whatever the band actually plays.
  • Runtime math and set summaries serialize into the URL, which makes it easy to drop a “latest plan” link into chat.
  • The Nextcloud upload webhook still feeds a “StickWithIt Clips” folder so finished videos move themselves into the archival spot once the status flips to done.
